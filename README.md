# task04-

Project Overview

This project presents an interactive business intelligence dashboard designed for business stakeholders to monitor key performance metrics such as Sales, Profit, and Growth.
The dashboard enables data-driven decision making through dynamic filters, KPI cards, and time-series analysis.

Built using Power BI / Tableau with a focus on clarity, usability, and business insights.

- Objectives

Provide a high-level overview of business performance

Track sales and profit trends over time

Identify top-performing and underperforming categories & regions

Enable interactive exploration using filters and slicers

Present insights in a visually appealing and professional format

- Key Performance Indicators (KPIs)

Total Sales

Total Profit

Profit Margin (%)

Sales Growth (%)

Number of Orders

Top Products by Revenue

- Tools & Technologies

Power BI / Tableau – Dashboard development

Microsoft Excel / CSV Dataset – Data source

PowerPoint – Summary presentation

GitHub – Version control & documentation

- Dataset

The dataset contains retail sales transaction data including:

Order Date

Region & State

Product Category & Sub-Category

Sales, Profit, Quantity

Customer Segment

Dataset Source (Kaggle):
- Superstore Sales Dataset

- Dashboard Features

✔ Interactive slicers (Date, Region, Category, Segment)
✔ KPI cards for quick business overview
✔ Time-series analysis using line charts
✔ Category & region-wise comparison
✔ Top products performance table
✔ Consistent color theme and layout
✔ Optional navigation menu for page switching

- Visualizations Used

Line Chart – Sales & Profit over time

Bar Chart – Sales by Category

Donut Chart – Profit distribution

Map (optional) – Regional performance

KPI Cards – Summary metrics

Table – Top products / customers

- Key Insights (Sample)

Sales show steady growth over recent periods

Technology category generates the highest profit

Certain regions consistently underperform and need strategic focus

Seasonal patterns observed in monthly sales trends
